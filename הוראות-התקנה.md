# MORALI — הוראות התקנה (5 דקות)

החבילה הזו מכילה את כל האתר + המתכנן החכם:

```
index.html      ← דף הבית
planner.html    ← עמוד המתכנן החכם (AI אמיתי)
api/plan.js     ← פונקציית שרת שמסתירה את מפתח ה-API
```

---

## שלב 1 — מפתח API של Anthropic (פעם אחת)

1. היכנסו ל- **console.anthropic.com** והירשמו עם המייל שלכם
2. בתפריט: **Billing** → הוסיפו אמצעי תשלום (אפשר להתחיל ב-$5)
3. בתפריט: **API Keys** → **Create Key** → תנו שם (למשל morali) → **Create**
4. **העתיקו את המפתח** (מתחיל ב- `sk-ant-`) ושמרו בצד — הוא מוצג פעם אחת בלבד

> חשוב: אל תשתפו את המפתח עם אף אחד ואל תדביקו אותו בשום קובץ.
> הוא נכנס רק להגדרות של Vercel (שלב 3).

---

## שלב 2 — העלאת הקבצים ל-GitHub

1. היכנסו למאגר שלכם: **github.com/moralidisplay-a11y/morali**
2. לחצו **Add file → Upload files**
3. גררו לחלון את **כל התיקייה** (את 3 הקבצים, כולל תיקיית api) —
   הגרירה של תיקייה שלמה שומרת על המבנה
4. לחצו על הכפתור הירוק **Commit changes**

ודאו שאחרי ההעלאה רואים במאגר:
- index.html
- planner.html
- api (תיקייה ובתוכה plan.js)

---

## שלב 3 — הגדרת המפתח ב-Vercel

1. היכנסו ל- **vercel.com** → בחרו את הפרויקט **morali**
2. למעלה: **Settings** → בתפריט הצד: **Environment Variables**
3. הוסיפו משתנה חדש:
   - **Name (Key):** `ANTHROPIC_API_KEY`
   - **Value:** הדביקו את המפתח מ-שלב 1
4. לחצו **Save**

---

## שלב 4 — פריסה מחדש

1. בפרויקט ב-Vercel: לשונית **Deployments**
2. ליד הפריסה האחרונה: תפריט שלוש נקודות **⋯ → Redeploy → Redeploy**

(אם חיברתם GitHub — כל Commit עתידי יתפרס אוטומטית, אבל אחרי
הוספת משתנה סביבה צריך Redeploy אחד ידני)

---

## שלב 5 — בדיקה

1. גלשו ל- `הכתובת-שלכם.vercel.app/planner.html`
2. מלאו פרטים, העלו תמונת קיר, לחצו "צרו לי תכנון חכם"
3. אמורים לקבל: ניתוח קיר, סקיצת הצבה, רשימת מוצרים עם מחירים

אם מופיעה שגיאה "חסר מפתח API" — חזרו לשלב 3 ובדקו שהשם
בדיוק `ANTHROPIC_API_KEY` ושעשיתם Redeploy.

---

## עדכון הקטלוג למוצרים האמיתיים שלכם

הקטלוג נמצא בקובץ **api/plan.js** בראש הקובץ (CATALOG).
לכל מוצר: id באנגלית, שם, קטגוריה, מחיר, רוחב/עומק במטרים,
ולאילו סוגי עסקים הוא מתאים.
עורכים → Commit → האתר מתעדכן אוטומטית.

## עלויות

כל תכנון של לקוח עולה בערך 1–4 סנט (אגורות בודדות).
ב-console.anthropic.com אפשר להגדיר תקרת הוצאה חודשית
(Billing → Limits) — מומלץ להתחיל מ-$10.
